### Programming language used
Python

### Required packages
Included in "requirements.txt" file. Some dependencies are not necessary but it is a working environment.

### How to produce the result
1. Download the files from Kaggle and store locally in "files" directory under the project directory structure.
2. Open "main.ipynb" as a jupyter notebook.
3. Run all cells.
4. With the final file, change the column names to "id,playtime_forever" manually (no space between commas and names".

### Result file
"training_result.csv"
